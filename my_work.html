QueroElo!

QueroElo! is a game services web-store. It was built using the Django Web-Framework. In conjunction to it, there is a system to keep track of all hired personnel, as well as the way the team comunicates, staff members get job assigments and a REST API for managing purchases. It is fully integrated with Slack and handles payments via the PicPay API. I sketched, built, deployed and maintain the complete project.
